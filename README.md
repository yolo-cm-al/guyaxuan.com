# 猜猜我是谁游戏

一个基于网页的猜谜游戏，玩家需要根据提示猜出隐藏的角色。

## 功能特点

- 动画人物角色猜测
- 提示系统
- 计分机制
- 响应式设计

## 在线演示

部署后可通过以下地址访问：
`https://你的用户名.github.io/仓库名称/`

## 文件说明

- `index.html` - 主页面文件
- `style.css` - 样式文件  
- `script.js` - JavaScript游戏逻辑

## 技术栈

- HTML5
- CSS3
- 原生JavaScript

## 如何运行

1. 克隆仓库到本地
2. 直接在浏览器中打开 `index.html` 文件
3. 或者部署到GitHub Pages在线访问

## 游戏规则

1. 根据角色类别和提示信息猜测角色名字
2. 每次猜测后会有相应的反馈
3. 限制5次猜测机会
4. 猜对获得积分，可以进入下一题
